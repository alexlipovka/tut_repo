Описание

Данный проект предназначен создан для студентов-градостроителей, чтобы познакомиться с СУБД PostGIS и Geoserver

Процесс

Для последующих операций необходимо наличие слоев с векторными геоданными.

  1. Открыть QGIS
  2. Создать подключение к БД PostGIS
  3. Создать новую схему
  4. Загрузить свои слои в созданную схему
  5. Открыть страницу с геосервером
  6. Добавить новое хранилище с указанием созданной схемы
  7. Опубликовать загруженные слои
